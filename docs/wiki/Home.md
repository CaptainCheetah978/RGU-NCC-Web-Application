# Welcome to the NCC RGU Cadet Management System Wiki! 

**Jai Hind!**

Welcome to the digital command center. Whether you are an ANO overseeing unit operations, a SUO managing the administrative workflow, or a developer maintaining the codebase, this Wiki is your comprehensive guide to mastering the Cadet Management System.

This system was built to modernize battalion administration-transitioning manual administrative tasks into a secure, high-performance, and offline-capable digital platform. Let's get started.

## ️ What's inside?

We have structured the documentation into three core manuals:

1. **[User Guide](User-Guide.md)**: The complete field manual for all ranks. From first-time logins and offline attendance marking, to thoughtfully archiving graduating cadets to the Alumni vault. If you need to know how a feature works, start here.
2. **[Developer Guide](Developer-Guide.md)**: The engineering blueprint. Read this to understand our offline synchronization queue, our database architecture, and how we achieve perfectly smooth scrolling with thousands of active cadets.
3. **[Troubleshooting & FAQ](Troubleshooting-and-FAQ.md)**: The diagnostic center. Before opening an issue, check here. If you encounter quirks with mobile ID card printing, or if you need to manually resolve a synchronization conflict, we've documented the solutions.
